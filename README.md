# Installation 

*Instructions and Rake file started from https://github.com/craveytrain/dotfiles/*

+ Clone the repo

```sh
git clone --recursive git@github.com:rubikzube/dotfiles.git
```

+ Get inside the repo

```sh
cd dotfiles
```

+ Run the Rakefile

```sh
rake install
```
